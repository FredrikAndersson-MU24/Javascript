    true

  3. Vad är den tekniska skillnaden ( förutom var, let ) på dessa två deklarationer?


    let name = 'Greta Thunberg';
    var name = 'Greta Thunberg';

    • let = re-assignable, block-scoped local variables. Cannot be redeclared. Can only be accessed after the place of declaration is reached (non-hoistable).

    • var = function-scoped or globally-scoped. Can be redeclared.

  4. Hur tar man reda på vad en variabel har för datatyp?


    Med typeof operatorn. T ex

    var name = 'Greta Thunberg';
    console.log(typeof name); // string
    var num = 10;
    console.log(typeof num); //number
